Atomic CSS

Atomic CSS, también conocido como Utility-First CSS, es una técnica que consiste en crear clases que aplican una única propiedad CSS. Estas clases se reutilizan para componer el diseño de una página. Uno de los framework CSS más conocidos que usan esta técnica es Tailwind CSS.

Por ejemplo, con Atomic CSS, en lugar de crear una clase específica para un componente, se utilizan clases más generales que describen una propiedad CSS. Por ejemplo, para aplicar display: flex a un elemento, usarías la clase d-flex:

.d-flex {
  display: flex;
}

Luego, puedes combinarlas:

Este es un ejemplo usando Atomic CSS.

<div class="d-flex justify-center align-center p-10 m-20 bg-light border">
  <p>Este es un ejemplo usando Atomic CSS.</p>
</div>
